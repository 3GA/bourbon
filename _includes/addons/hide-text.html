hide-text is an image replacement mixin. Itâ€™s based off the HTML5 Boilerplate image replacement.

This image replacement technique is an alternative to the text-indent technique and has performance and other benefits.

A height declaration is required for this to function, which can be passed as an argument (default is 1em).

{% highlight scss %} .element { @include hide-text; background-image: url(logo.png); } .element { @include hide-text(2em); // passing an optional height argument } {% endhighlight %}

{% highlight css %} .element { height: 1em; line-height: 1.5; overflow: hidden; background-image: url(logo.png); } .element::before { content: ""; display: block; width: 0; height: 100%; } {% endhighlight %}
