HiDPI Media Query

View source View spec

The hidpi Meda Query will allow you to generate a media query that targets HiDPI devices. It accepts an optional ratio argument, default ratio is 1.3. Find my device pixel ratio.

{% highlight scss %} @include hidpi(1.5) { width: 260px; } {% endhighlight %}

CSS Output

{% highlight scss %} @media only screen and (-webkit-min-device-pixel-ratio: 1.5), only screen and (min--moz-device-pixel-ratio: 1.5), only screen and (-o-min-device-pixel-ratio: 1.5/1), only screen and (min-resolution: 144dpi), only screen and (min-resolution: 1.5dppx) { width: 260px; } {% endhighlight %}
