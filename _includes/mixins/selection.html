Selection

View source View spec

Outputs vendor-prefixed selection selectors for styling. Can be included at base level.

{% highlight scss %} @include selection { background-color: #fff; } p { @include selection { background-color: #000; color: #fff; } } {% endhighlight %}

CSS Output

{% highlight scss %} ::selection { background-color: #fff; } ::-moz-selection { background-color: #fff; } p::selection { background-color: #000; color: #fff; } p::-moz-selection { background-color: #000; color: #fff; } {% endhighlight %}
